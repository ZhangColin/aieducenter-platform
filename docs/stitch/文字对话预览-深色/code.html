信息熵与交叉熵：信息论中的核心概念

1. 信息熵 (Information Entropy)

信息熵由克劳德·香农于1948年提出，用于量化随机变量的不确定性。对于一个离散随机变量 $X$，其概率分布为 $P(x)$，信息熵定义为：

H(X) = -\sum_{x \in \mathcal{X}} P(x) \log_2 P(x)

其中 $\mathcal{X}$ 是 $X$ 的取值空间。当使用自然对数时，单位为纳特 (nats)；使用以2为底的对数时，单位为比特 (bits)。

1.1 性质

  • 非负性: $H(X) \geq 0$
  • 确定性: 当某个 $P(x)=1$ 时，$H(X)=0$
  • 上界: 对于有 $n$ 个取值的均匀分布，熵取最大值 $\log_2 n$

2. 交叉熵 (Cross Entropy)

交叉熵衡量两个概率分布 $P$ 和 $Q$ 之间的差异。对于同一个随机变量，真实分布为 $P$，预测分布为 $Q$，交叉熵定义为：

H(P, Q) = -\sum_{x \in \mathcal{X}} P(x) \log_2 Q(x)

2.1 与 KL 散度的关系

交叉熵可以分解为信息熵与 KL 散度之和：

H(P, Q) = H(P) + D_{KL}(P \parallel Q)

在机器学习中，由于真实分布 $P$ 的熵 $H(P)$ 是常数，因此最小化交叉熵等价于最小化预测分布与真实分布之间的 KL 散度。
